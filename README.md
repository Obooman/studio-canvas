# Canvas Studio

一个基于 [tldraw](https://tldraw.dev) 的极简画布应用，专为录制 YouTube 演示视频而打造。界面采用暗色主题，隐藏了 tldraw 默认 UI，仅保留键盘快捷键操作，让录屏画面保持干净、聚焦。

## 功能特性

- **纯净画布**：隐藏默认工具栏，无边框、无阴影，适合录屏展示
- **暗色模式**：深色背景的画布与组件，减少录制时的视觉干扰
- **键盘驱动**：所有绘图工具通过快捷键切换，无需鼠标点击工具栏
- **网页嵌入**：直接粘贴 URL 即可在画布中嵌入可交互的网页
- **演示模式**：一键进入只读演示模式，嵌入内容可滚动交互
- **激光笔**：演示模式下按住快捷键即可使用激光笔标注
- **本地持久化**：画布内容自动保存在浏览器本地

## 快速开始

```bash
npm install      # 安装依赖
npm run dev      # 启动开发服务器
npm run build    # 构建生产版本到 dist/
npm run preview  # 预览构建产物
```

开发服务器启动后，在浏览器中打开终端显示的地址即可。

## 快捷键

### 绘图工具

| 按键 | 工具 |
| --- | --- |
| `V` | 选择 |
| `H` | 平移（画布拖动） |
| `D` | 画笔 |
| `R` | 矩形 |
| `O` | 椭圆 |
| `T` | 三角形 |
| `L` | 直线 |
| `A` | 箭头 |
| `X` | 文字 |

### 视图与演示

| 按键 | 功能 |
| --- | --- |
| `⌘/Ctrl` `P` | 切换演示模式 |
| `⌘/Ctrl` `+` / `=` | 放大 |
| `⌘/Ctrl` `-` | 缩小 |
| `⌘/Ctrl` `0` | 重置缩放 |
| 滚轮 | 平移画布 |

### 演示模式专用

| 按键 | 功能 |
| --- | --- |
| 按住 `L` | 激光笔 |
| 按住 `Space` | 临时穿透嵌入内容拖动画布 |

## 嵌入网页

在画布上直接粘贴（`⌘/Ctrl` `V`）任意 `http(s)` 链接，即可在视口中央创建一个可交互的网页嵌入框。进入演示模式后，嵌入的网页可以正常滚动和交互。

## 技术栈

- [React 19](https://react.dev)
- [tldraw 5](https://tldraw.dev) — 画布引擎
- [Vite](https://vite.dev) — 构建工具

```

  styles.css   # 暗色主题与画布样式覆盖
```
